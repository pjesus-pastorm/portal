Accesibilidad

  1. Inicio
  2. Accesibilidad

Las páginas web de este portal cumplen con los estándares de la W3C, según se establece en el Real Decreto 1112/2018, de 7 de septiembre, sobre accesibilidad de los sitios web y aplicaciones para dispositivos móviles del sector público.

Concretamente este portal cumple con todos los requisitos de Prioridad 1 y Prioridad 2 de la Norma UNE 139803:2004 y han sido comprobados con un análisis manual de la accesibilidad a través de diferentes herramientas semi-automáticas.


Ventajas

Las ventajas que esto conlleva son, entre otras son las siguientes:

  • Facilitar el acceso de las personas independientemente de su condición física, cognitiva o de su entorno.
  • Permitir el acceso con diferentes agentes de usuario. Incluir contenidos claros y bien estructurados.
  • Mejorar la navegación y la experiencia del usuario.

Medidas adoptadas

Entre las medidas adoptadas se encuentran las siguientes:

  • Orden lógico de presentación de los elementos en pantalla y de los contenidos en el código.
  • Uso de los estándares del W3C. Gramática XHTML y CSS.
  • Pautas de Accesibilidad para el Contenido Web (WCAG-2 nivel AA).

Teclas de acceso (accesskey)

Las teclas de acceso son un dispositivo de navegación que le permite navegar por este sitio web utilizando el teclado de una manera rápida. Mediante ellas se enlaza con las principales secciones y utilidades del portal.

  • Tecla 1 Página inicial
  • Tecla 2 Sección “El Ministerio”
  • Tecla 3 Sección “Información Destacada”
  • Tecla 4 Sección “Tu Administración”
  • Tecla 5 Sección “Sistema Universitario”
  • Tecla 6 Sección “Estadísticas”
  • Tecla 7 Sección “Prensa”

El uso de las teclas de acceso depende del navegador del usuario y del sistema operativo. Algunos casos son:

Navegador Windows Mac Linux
Chrome [Alt] + accesskey [Control] + [Alt] + accesskey [Alt] + accesskey
Edge [Alt] + accesskey n/a n/a
Internet Explorer [Alt] + accesskey n/a n/a
Firefox [Alt] + [Shift] + accesskey [Control] + [Alt] + accesskey [Alt] + [Shift] + accesskey
Safari [Alt] + accesskey [Control] + [Alt] + accesskey n/a
Opera Opera 15 o superior: [Alt] + accesskey Opera 12.1 o anteriores: [Shift] + [Esc] + accesskey
